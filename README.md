# AtmosIQ – Smart Weather Forecasting & AI Trip Planner

## 🚀 Overview
AtmosIQ is a full-stack MERN application that provides real-time weather updates along with an AI-powered trip planner.

## 🛠️ Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Machine Learning
- NLP

## ✨ Features
- Real-time weather forecasting
- NLP-based trip planner
- Interactive UI
- API integration
